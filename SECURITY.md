# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Axonium, **do not create a public issue**.

Contact us directly at: **security@axonium.fr**

### Information to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

### Response Time

- Acknowledgment: 48 hours
- Initial assessment: 7 days
- Fix: depending on severity
